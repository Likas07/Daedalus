# Forge-Daedalus Port

This directory collects planning materials for design and implementation work where Daedalus deliberately ports useful ideas from ForgeCode.

Current subdirectories:
- `agent-role-redesign/` — role architecture, prompt redesign, runtime migration notes, and completion status
- `tooling-port/` — search/tool surface port planning, including `sem_search` and `fs_search`

Useful entry points:
- `implementation-status.md` — live milestone implementation tracker
- `agent-role-redesign/current-state-and-next-steps.md` — current implemented runtime state of the role redesign
- `agent-role-redesign/remaining-work-checklist.md` — redesign completion checklist / completion gates
- `agent-role-redesign/agent-system-redesign-product-spec.md` — implemented product/runtime shape for the redesign
