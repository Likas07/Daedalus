# Executing plans

Use this skill when implementing tasks from a written plan.

- Follow the active task scope; do not expand into adjacent plan items without approval.
- Inspect the existing code first, then make minimal intentional changes.
- Keep progress aligned to the plan and note blockers as soon as they appear.
- Verify each task with the plan's targeted commands before reporting completion.
