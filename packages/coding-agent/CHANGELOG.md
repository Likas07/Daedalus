# Changelog

## [0.1.0] - 2026-04-10

### New

- Initial release of Daedalus, a self-contained fork of Pi
- Bun-native runtime with source-first execution and single-binary compilation
- Baked-in extensions: safety guards, workflow tools, UI enhancements
- Gold/artisan dark theme inspired by Greek mythology
- CLI available as `daedalus` and `dae`
- Config directory at `~/.daedalus/`
- Extension system with `"daedalus"` manifest key (backwards-compatible with `"pi"`)
